# Trilium Web

Static marketing website for `triliumweb.com`, positioned around fast monthly website plans for trades businesses.

Current offer:

- Free website preview before signup
- Monthly only, no setup fee
- Public tiers: Online at $9/mo, Trades at $49/mo, Lead at $99/mo
- Add-ons for extra pages, local SEO help, Google Business Profile cleanup, and priority support
- Personal maintenance support

## Files

- `index.html` - homepage
- `process.html` - service process
- `pricing.html` - plan overview
- `work.html` - preview categories
- `contact.html` - preview request form
- `thanks.html` - form submission thank-you page
- `assets/styles.css` - shared styling
- `assets/site.js` - mobile navigation and header behavior

## Form submissions

The preview request form posts to FormSubmit at `hello@triliumweb.com`.
The first live submission will send an activation email to that inbox; confirm it once to enable delivery.

Open `index.html` directly in a browser, or serve the folder with any static file server.
